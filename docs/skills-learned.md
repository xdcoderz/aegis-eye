# Skills Built Through AegisEye

## Computer Vision

- Open webcam and video-file streams with OpenCV.
- Convert frames to grayscale for detector input.
- Apply Haar cascade face detection and tune scale, neighbor, and minimum-size parameters.
- Work with bounding boxes and safely crop image regions.
- Apply privacy blur before writing processed output.
- Encode redacted frames deterministically enough for integrity hashing.
- Separate detection quality from cryptographic evidence quality.
- Understand why accuracy evaluation is required before claiming privacy guarantees.

## Applied Cryptography

- Use SHA-256 for content integrity.
- Use Ed25519 keys, signatures, and verification.
- Persist private edge identity without transmitting the private key.
- Pair and compare public device keys.
- Understand why canonical serialization is required before hashing structured data.
- Build and audit previous-hash chains.
- Distinguish tamper evidence from immutability.
- Identify the need for independent chain-head anchoring and hardware key custody.

## Backend Engineering

- Design validation-first REST ingestion APIs.
- Use Spring Boot controllers, services, exception handling, and transactions.
- Query and update PostgreSQL with Spring JDBC.
- Enforce uniqueness, allowed states, and hash formats in the database.
- Return problem details and conflict responses.
- Build read models for streams and ledger records.
- Keep a demo-only capability configurable and scope-restricted.

## Distributed Systems

- Define a cross-language JSON contract shared by Python and Java.
- Preserve sequence continuity across edge and server boundaries.
- Reject replayed or branching records.
- Handle trust on first use and recognize its enrollment limitations.
- Separate source media, evidence metadata, and verification state.
- Reason about race conditions, key rotation, deletion, and external anchoring.

## Frontend and Product Design

- Turn low-level integrity checks into an operator-readable workflow.
- Display stream state, evidence records, and precise failure reasons.
- Build a controlled adversarial test into the demonstration.
- Design a fallback path for unreliable judging environments.
- Scope an MVP around one defensible claim.
- Communicate proven behavior separately from production aspirations.

## Testing and Delivery

- Unit-test deterministic canonicalization, signatures, persistent keys, and hash links.
- Compile and test independent Python and Java components.
- Run a full local edge-to-API-to-database-to-verifier smoke test.
- Write a reproducible demo runbook, threat model, architecture, API reference, and submission brief.

## Portfolio Value

AegisEye demonstrates more than framework familiarity. It shows the ability to combine computer vision, security, backend persistence, cross-language contracts, user-facing verification, threat modeling, and honest product scoping into one working system. That combination is valuable for computer-vision, backend, security, applied AI, and systems-engineering roles.
